This are some of the basic Git commands:

cd              Takes you back to library
ls              Show the folders/library inside the folder
pwd             Shows the directory tree
mkdir           Makes a new folder
git clone git@github.com:DaneyLui/Learning.git      Clones your github SSH repository to your local machine. 
git remote -v   Displays the URL of the repository
touch hello.txt     Makes a new file
git status      Shows the status of things
git add hello.txt   Adds the file to the statiging area in Git.    
git commit  -m "Add hello.txt"      Add the commit to 
git log           Logs the commit into git
git push
git push origin main        Push all the commits to github
